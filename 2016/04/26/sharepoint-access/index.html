文件馆访问权限设定

sharepoint 2007
文件馆访问的最低权限级别包括以下权限项目：
网站权限：检视页面、开启
清单权限：检视项目、检视应用程式页面

sharepoint 2013
文件馆访问的最低权限级别包括以下权限项目：
网站权限：检视页面、开启
清单权限：检视项目、開啟項目、检视应用程式页面

sharepoint 2013
文件馆新增、編輯、刪除权限级别在以上基礎下包括以下权限项目：
网站权限：使用用戶端整合功能
清单权限：新增項目、編輯項目、刪除項目

sharepoint 2013
install on server 2012 r2
prerequisiteinstaller配置IIS等功能始終報錯，忽略直接安裝即可
安裝WindowsServerAppFabricSetup_x64時失敗，需修改系統環境變量PSModulePath中的引號
