Python xlwt

xlwt模块

导入xlwt模块

import xlwt

新建Excel中sheet
w = xlwt.Workbook()
ws = w.add_sheet(sheetname, cell_overwrite_ok=True)

cell_overwrite_ok=True设置Excel内容可覆写

设置格式
style = xlwt.XFStyle()
font = xlwt.Font()
font.name = u'新細明體'
style.font = font
在Excel对应位置写入内容

ws.write(col, row, u'\u25cf', style)

保存Excel

w.save(filepath)
